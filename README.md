# Cyber System Interface

Projeto Front-End inspirado em interfaces cyberpunk e sistemas HUD futurista.

## 📌 Objetivo

Desenvolver uma interface interativa utilizando HTML, CSS e JavaScript, aplicando conceitos de:

- Estrutura semântica
- Estilização avançada
- Animcações 
- Manipulação do DOM
- Efeitos visuais cyberpunk
- Interação com Git e GitHub

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts (Orbitron)

---

## ⚡ Funcionalidades

- Terminal animado com digitação letra por letra
- Barra de carregamento dinâmica
- Relógio digital em tempo real
- Interface neon cyberpunk
- Cards interativos
- Formulário estilizado
- Footer com redes sociais
